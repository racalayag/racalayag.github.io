A definition list

HTML
Hypertext Markup Language - Markup that tells a browser how to display a webpage and it's contents. HTML is not a programming language. It origninated from SGML before Tim Berners-Lee created HTML to make the sharing of research papers online easier.
CSS
Cascading Style Sheet - CSS is not a programming language, but it deals with the visuals of a webpage. This typically includes: colours, fonts, text size, etc.
FTP
File Transfer Protocol - A method of transferring (downloading, uploading) files from computer to computer. An example is you, the user, downloading an image from Google.
SSH
Secure Shell - Network communication that provides a secure connection between two endpoints. This usually entails the usage of asymmetric keys to lock files and allows only the users involved to access the contents of the message. There are usually two sets of keys involved, public and private.
